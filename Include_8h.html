Inserts a "metadata" - a key/value string pair into the stream.

NOTE: The metadata are modelled as MKV tags and are not immediately put into the stream as it might break the fragment. This is a limitation of MKV format as Tags are level 1 elements. Instead, they will be accumulated and inserted in-between the fragments and at the end of the stream.

MKV spec is available at: https://matroska.org/technical/specs/index.html

Putting a "persistent" metadata will result in the metadata being inserted before every fragment. The metadata can be changed by calling this function with the same name and a different value.
