Decodes hex data. Calling the function with NULL output buffer will result in just the buffer size calculation. NOTE: pInputData should be NULL terminated IMPLEMENTATION: We will ignore the last character if the number is odd. Will return an error on non-hex chars. Will process the upper and lower case.
